## ✨ Features

- **Search by dish name** — find pasta, curry, tacos and more
- **Search by ingredients** — enter what you have at home
- **Filter by cuisine** — Italian, Indian, Japanese, Thai and more
- **Filter by diet** — Vegetarian, Vegan, Gluten Free, Keto, Paleo
- **Filter by time** — under 15, 30, or 60 minutes
- **Sort** — by popularity, quickest, or lowest calories
- **Recipe modal** — full ingredients list and step-by-step instructions
- **Save favourites** — heart any recipe, view all saved on the Favourites page
- **Dark mode** — toggle with one click, preference saved
- **Trending tags** — quick search popular dishes
- **Responsive** — works on all screen sizes

## 📁 Project Structure

```
recipe-finder/
├── index.html           ← Main search page
├── .nojekyll            ← GitHub Pages fix
├── css/
│   ├── style.css        ← Main styles
│   └── darkmode.css     ← Dark theme variables
├── js/
│   ├── theme.js         ← Dark mode toggle
│   ├── recipes.js       ← Recipe database (15 recipes)
│   └── app.js           ← Search, filter, render logic
└── pages/
    └── favourites.html  ← Saved recipes page
```

## 🚀 Run Locally

1. Open folder in VS Code
2. Right-click `index.html` → **Open with Live Server**
3. Opens at `http://127.0.0.1:5500`
